Like I said before, I am currently taking a class on html and css. This page will give you more background informtion on HTML AND CSS

a tag is the basic building block for most websites.

one of the most basic item in HTML is called a tag. A tag a function made up of a "greater than" and "less than" sign

Some of the most basic tags include:

  • The p tag (p) The p tag creates a paragraph that you can write large amounts of text in
  • The header tag (h1). The header tag creates a header, you can create headers of different sizes by changing the number next to the h.
  • The a tag (a or a href). The a tag creates a link to website that will activate when you click on it
  • The image tag (img or img src). The image tag will make an image appear, in order for the image tag to work, you have to put the link to the image.

Another basic part of HTML is called an attribute. An attribute tells you more about the HTML.

  • For example, an "a" tag needs a "href" for the link to work
  • Also, the "img" tag also needs "src" for the image to appear
  • one of lesser known are the width and the height, which specify how big the image will need to be

CSS is used to syle HTML, using selectors (which are the tags in HTML).

It easier to use CSS to change overall fonts and font color, mostly because it requires about 3 or 4 lines of code
